# 🎯 Objective

इस Chapter में हम सीखेंगे—

- README.md क्या होता है?
- Enterprise Project में README इतना Important क्यों होता है?
- README में क्या-क्या होना चाहिए?
- 40 Chapters वाले Project का README कैसे Design करें?
