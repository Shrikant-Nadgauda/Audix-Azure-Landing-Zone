# 🎯 Objective

---

# 🤔 GitHub Banner क्या होता है?

जब कोई Visitor आपकी Repository Open करता है...

तो सबसे पहले उसे

```text
README.md
```

दिखाई देता है।

README का सबसे पहला भाग

**GitHub Banner**

होता है।

यही आपकी Repository की First Impression बनाता है।

---
